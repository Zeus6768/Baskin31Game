# Baskin31Game
🍨 네트워크 과제로 만든 베스킨라빈스31 게임입니다. 

## 규칙
- 5명 모두 참가해야 게임을 시작할 수 있습니다.
- 플레이어들은 번갈아가며 숫자를 외칩니다. 
- 이때, `직전 플레이어가 외친 숫자 +1 ~ 직전 플레이어가 외친 숫자 +3`의 범위 내의 숫자를 외칠 수 있습니다. 
- 숫자는 31까지 있습니다. 
- 마지막에 31을 외친 플레이어가 패배합니다.
